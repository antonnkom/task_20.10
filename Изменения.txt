---------------------------------------------------------------------------------------------------------------------
Принцип DRY
---------------------------------------------------------------------------------------------------------------------
1. Исправелно разметка навигационного блока в шапке и в футере, так, что этот фрагмент верстки можно использовать и в шапке и в футере, заменены классы.

2. Все теги button, input переписаны так, что используются повторно.

3. Удалены лишние вложенные теги, например 
<div class="container">
	<div class="control-cntainer">...</div>
</div>

изменено на:
<div class="container control-cntainer">...</div>

---------------------------------------------------------------------------------------------------------------------
Принцип KISS
---------------------------------------------------------------------------------------------------------------------
к тегам button и input добавлены Модификаторы, изменяющие их внешний вид. Прописаны стили Модификаторов. Стили заголовков у кнопок и полей не используются. 
.title-white и .bg-color удалён из input'ов

---------------------------------------------------------------------------------------------------------------------
Принцип YAGNI
---------------------------------------------------------------------------------------------------------------------
Удален JavaScript-код, реализующий переключение вкадок, т.к. в ТЗ на верстку не требовалось реализовывать функционал переключения вкладок. (В старой версии данный скрипт присутствует, но не подключен, но если его подключить, то переключение вкладок работает).

---------------------------------------------------------------------------------------------------------------------
Принцип SOLID
---------------------------------------------------------------------------------------------------------------------
В CSS SOLID был реализован следующим образом, например:
в header и в footer в навигации класс

.nav-container {
    display: flex;
    height: 83px;
    justify-content: space-between;
    align-items: center;
}

был разбит на два класса:

Отвечающий за высоту блока
.nav-container {
    height: 83px;
}

Отвечающий за позиционирование внутри блока
.nav-wrapper {
    display: flex;
    justify-content: space-between;
    align-items: center;
}
